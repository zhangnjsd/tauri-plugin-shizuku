## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-get-status`
- `allow-request-permission`
- `allow-open-shizuku`
- `allow-run-adb-command`
- `allow-get-system-property`
- `allow-list-packages`
